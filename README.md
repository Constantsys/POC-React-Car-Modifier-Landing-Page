# Car Modify Company – POC Landing Page

## About
This project is a **Proof of Concept (POC)** landing page website built for a **Car Modification Company** using **ReactJS** and **Bootstrap**.

The landing page showcases car modification services, customer reviews, locations, and scheduling information with a modern, responsive UI.  
It is designed as a marketing-focused frontend to validate layout, branding, and user experience.

---

## Tech Stack
- **ReactJS** – Frontend framework
- **Bootstrap 5** – Responsive UI framework
- **SCSS / CSS** – Custom styling
- **JavaScript (ES6+)**
- **HTML5**

---

## Features
- Responsive landing page layout
- Hero section with call-to-action
- Services overview (car modification services)
- Customer reviews/testimonials
- Location information
- Schedule / appointment section
- Reusable UI components
- Mobile-friendly design
